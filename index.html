Buffon's Matches: approximating PI with simulated matches

We can approximate PI by throwing some matches on the floor. This simulation throws thousands of matches on your screen. In the background there are some vertical lines equally spaced from each other. Whenever a match crosses such a line it will be in a nice color, otherwise it will be gray. When we count all the matches thrown and divide it by the matches in a nice color, we get PI!
This was inspired by an older Numberphile video.
